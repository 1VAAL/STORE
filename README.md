# أنمي ستور — متجر كامل بحسابات ورتب

موقع متجر إلكتروني (فيجرز، بلوشيز، ملابس، إكسسوارات...) بتصميم فاتح أزرق/بنفسجي،
مع نظام حسابات حقيقي (تسجيل دخول/إنشاء حساب بإيميل وباسورد) ونظام رتب:

| الرتبة | بالعربي | الصلاحيات |
|---|---|---|
| `member` | عضو | تصفح، شراء، مفضلة |
| `publisher` | ناشر | كل حاجة فوق + نشر عروض في صفحة العروض |
| `admin` | أدمن | كل حاجة فوق + تغيير رتب المستخدمين من لوحة الأدمن |

**الإيميل `osamabrazo@gmail.com` بيبقى أدمن تلقائيًا من أول ما يسجل حساب.**
باقي الرتب الأدمن هو اللي بيديها يدويًا من لوحة التحكم.

⚠️ **مهم:** التحقق من الرتب مش بيعتمد على كود الموقع (JavaScript) فقط، لأن أي حد يقدر
يفتح Developer Tools ويغيّره عنده. التحقق الحقيقي موجود في ملف `firestore.rules`
اللي بيتحقق من هوية المستخدم فعليًا على سيرفرات Google — وده اللي بيمنع أي حد يزوّر
نفسه كأدمن حتى لو عدّل الكود عنده.

---

## 1. إنشاء مشروع Firebase (5 دقايق)

1. روح https://console.firebase.google.com وسجّل دخول بأي حساب Google بتاعك (ده حسابك إنت كصاحب الموقع بس، الزوار مش هيشوفوه خالص).
2. دوس **إضافة مشروع (Add Project)** وسمّيه زي ما تحب.
3. من القائمة الجانبية: **Build → Authentication → Get Started**
   - فعّل طريقة **Email/Password** بس (مفيش Google Sign-in هنا خالص).
4. من القائمة الجانبية: **Build → Firestore Database → Create Database**
   - اختار وضع **Production mode**.
5. من ⚙️ **Project Settings → عام (General)** → انزل لـ "Your apps" ودوس أيقونة الويب `</>`
   - هيديك object اسمه `firebaseConfig` فيه القيم المطلوبة.

## 2. تعبئة الإعدادات في الموقع

افتح `js/firebase-config.js` واستبدل القيم بالقيم اللي جبتها من الخطوة اللي فاتت:

```js
const firebaseConfig = {
  apiKey: "...",
  authDomain: "...",
  projectId: "...",
  storageBucket: "...",
  messagingSenderId: "...",
  appId: "...",
};
```

## 3. رفع قواعد الأمان (خطوة إجبارية جدًا)

1. في Firebase Console: **Build → Firestore Database → Rules**
2. امسح اللي موجود، وانسخ محتوى ملف `firestore.rules` كامل والصقه مكانه.
3. دوس **Publish**.

من غير الخطوة دي، أي حد هيقدر يعدّل رتبته لنفسه ويبقى أدمن!

## 4. تجربة الموقع محليًا

افتح `index.html` مباشرة في المتصفح (دبل كليك) — هيشتغل فورًا، جرّب:
- اعمل حساب بإيميل `osamabrazo@gmail.com` → هتلاقي نفسك أدمن تلقائيًا وتقدر تفتح "لوحة الأدمن"
- اعمل حساب بإيميل تاني عادي → هيبقى "عضو"، وتقدر من لوحة الأدمن ترقّيه لـ "ناشر" عشان ينشر عروض

## 5. رفع الموقع على GitHub Pages (استضافة مجانية)

1. اعمل repository جديد على GitHub وارفع كل ملفات المشروع (`index.html`, مجلد `css`, مجلد `js`).
2. من **Settings → Pages** فعّل النشر من البرانش `main` والمجلد `/(root)`.
3. الموقع هيبقى شغال على: `https://username.github.io/repo-name/`

⚠️ لازم بعد ما ترفع الموقع، تروح لـ Firebase Console → Authentication → Settings →
**Authorized domains** وتضيف الدومين بتاع GitHub Pages بتاعك، وإلا تسجيل الدخول مش هيشتغل.

## 6. تعديل المنتجات

كل المنتجات والتصنيفات في ملف واحد: `js/data.js` — عدّل الاسم/السعر/الصورة/الوصف من هنا.

## هيكل المشروع

```
index.html              الهيكل الكامل لكل صفحات الموقع
css/style.css           التصميم
js/firebase-config.js   إعدادات Firebase (لازم تعبيها)
js/data.js              المنتجات والتصنيفات
js/auth.js              تسجيل الدخول / الحسابات / الرتب
js/store.js             المتجر، البحث، السلة، الدفع
js/publish.js           نشر العروض + لوحة الأدمن
firestore.rules         قواعد الأمان (ارفعها في Firebase Console)
```
